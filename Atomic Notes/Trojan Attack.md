# Trojan Attack

> [!summary] 
> An attack where a user inadvertently introduces malicious code into the organization and triggers it behind defensive walls.

Types of trojan attacks:
- Exploit Trojans
- Downloader Trojans
- Ransom Trojans
- Backdoor Trojans
